# Learning_Python_Day15

Today's topic is modules and imports!

**Goal:** Understand Python's module system and how to organize code across multiple files

- **task1:** Create a `utilities.py` module with helper functions (calculator functions, string manipulation, etc.)

- **task2:** Import and use standard library modules like `datetime` (for timestamps) and `random` (for number generation)

- **task3:** Build a `main.py` file that imports and uses your custom module

- **task4:** Practice different import styles:
  - `import module`
  - `from module import function`
  - `import module as alias`

**mini-project:**
Multi-File Calculator:

Create a calculator program split into multiple files:

- `math_operations.py` → basic operations (+, -, *, /)  
- `advanced_operations.py` → power and square root  
- `calculator.py` → main program that imports both modules
